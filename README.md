# VocabHub
A Real Time Dictionary App

VocabHub is a clean, responsive web application that allows users to look up word definitions instantly.
This project was built to practice handling JavaScript and REST API integration using modern web standards.

🚀 Features
Instant Search: Fetches real-time data from the Free Dictionary API.
Results: Displays definitions
Fully Responsive UI: Designed with a Mobile-First approach using CSS Flexbox to ensure the search bar and buttons work perfectly on all screen sizes.
Error Handling: Provides user-friendly feedback when a word is misspelled or not found in the database.
Optimized Touch Targets: Buttons are designed with a minimum height for better accessibility on mobile devices.

🛠️ Tech Stack
Frontend: HTML5, Semantic CSS3 (Flexbox & Media Queries)
Scripting: JavaScript (ES6+)
API: RESTful API integration using fetch()
Deployment: GitHub Pages / Vercel

🧠 What I Learned
Building this project helped me master several core concepts:
1) DOM Manipulation: Dynamically updating the page content based on API responses.
2) Responsive Layouts: Solving the "button size" and "input overflow" issues using relative units (rem, %) and Media Queries.
3) API Consumption: Understanding JSON structures and how to extract specific nested data.
